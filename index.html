solarized-dark-pygments by gthank

A Pygments style based on the dark background variant of Solarized.

This is a simple style for Pygments that is inspired by the Solarized project by Ethan Schoonover.

This theme is for the dark background variant of Solarized. If you're looking for a light background version, try john2x's solarized-pygment.

Install

This is just a Style, not a plugin, so you need to add it to the styles directory of your Pygments distribution.
